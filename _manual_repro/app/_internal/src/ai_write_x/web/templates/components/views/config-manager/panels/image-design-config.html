使用说明
  • 修改后需点击右上角保存设置才会写入配置文件。
  • 「设计器主题」在 常规设置 → 应用外观 中配置，与本页 HTML 样式无关。
  • 勾选「原始 HTML 样式」后，下方参数仅保留备查，不会参与生成。
  • 文章库中可用「页面设计」单独微调单篇，仍以本页为默认基准。
