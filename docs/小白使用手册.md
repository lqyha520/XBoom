# 小爆来咯小白使用手册

> 这份手册写给第一次使用的小白用户。你不需要懂代码，只要照着步骤完成配置，就可以用它生成公众号文章、配图、保存草稿，后面再按需发布到公众号。

---

## 1. 这是什么软件

**小爆来咯** 是一款本地电脑上运行的 AI 图文创作工具，主要用来做公众号内容生产。

它可以帮你做这些事：

- 输入一个选题，让 AI 自动写一篇文章。
- 不输入选题，让软件从热点里找题材。
- 自动生成封面图、正文配图，并套用排版模板。
- 把生成好的文章保存到文章库。
- 配好微信公众号后，把文章上传到公众号草稿箱。
- 设置定时任务，到点自动写稿或发稿。

新手建议先跑通这条最简单路线：

```text
启动软件 -> 配置大模型 API -> 生成第一篇文章 -> 在文章库查看 -> 手动复制或上传草稿
```

---

## 2. 第一次使用前要准备什么

### 必须准备

**大模型 API Key**

这是 AI 写文章用的账号密钥。没有它，软件无法生成正文。

你可以选择其中一家服务：

- DeepSeek：https://platform.deepseek.com
- 通义千问/阿里云百炼：https://bailian.console.aliyun.com
- OpenRouter：https://openrouter.ai
- 硅基流动：https://siliconflow.cn
- 魔搭 ModelScope：https://modelscope.cn

拿到的 Key 通常长得像 `sk-xxxxxx`。不同平台格式不完全一样，正常。

### 建议准备

**图片生成 API Key**

它负责封面图和正文插图。不配置也能先试写文章，但可能没有图，或者只能用随机图。

常见选择：

- Picsum：不用 Key，适合先测试流程。
- 阿里百炼/通义万相：需要 DashScope Key。
- 魔搭 ModelScope：需要平台 Token。
- ComfyUI：适合本机已经会用 ComfyUI 的用户。

### 发布公众号时才需要

**微信公众号 AppID 和 AppSecret**

只有当你想把文章直接上传到公众号草稿箱，或者做自动发布时，才需要配置。

新手建议：先不要急着自动群发，先上传到**草稿箱**，再去公众号后台人工预览。

## 3. 启动软件

### 如果你用的是安装包

1. 安装完成后，在桌面或开始菜单找到 **小爆来咯**。
2. 双击打开。
3. 第一次启动可能会检查或安装 **Microsoft Edge WebView2 Runtime**，这是桌面窗口需要的组件，等待完成即可。
4. 看到软件主界面后，就可以开始配置。

### 如果你用的是项目源码

在项目根目录可以双击：

- `启动小爆来咯.bat`
- 或 `启动.bat`

如果桌面窗口打不开，可以试试：

- `启动-浏览器.bat`

浏览器模式和桌面模式功能基本一致。

---

## 4. 认识左侧菜单

打开软件后，主要看左侧导航栏：

| 菜单 | 什么时候用 |
|---|---|
| 内容生成 | 写文章的地方，新手最常用 |
| 文章库 | 查看、搜索、发布、删除已生成文章 |
| 模板库 | 管理文章排版模板 |
| 选题中心 | 获取热点、找选题 |
| 运行中心 | 设置自动化任务、定时写稿 |
| 设置 | 配置大模型、图片生成、公众号等 |

新手第一天重点只用三个地方：

```text
设置 -> 内容生成 -> 文章库
```

---

## 5. 第一步：配置大模型 API

这是最重要的一步。

### 进入位置

左侧菜单点击：

```text
设置 -> 大模型API
```

如果你没看到二级菜单，点一下 **设置** 右边的小箭头展开。

### 配置步骤

1. 在 **配置厂商** 下拉框里选择你购买或开通的平台，比如 DeepSeek、Qwen、OpenRouter。
2. 找到 **API KEY** 输入区域。
3. 点击添加，把你的 Key 粘贴进去。
4. 如果有多个 Key，可以一行放一个。
5. 在 **模型** 里选择一个可用模型。
6. 点击 **测试连接**。
7. 测试成功后，点击 **设为当前使用**。
8. 最后一定要点右上角 **保存设置**。

### 怎么知道配置成功

满足下面三个条件就可以：

- 测试连接显示成功。
- 页面提示当前使用的是你选择的厂商。
- 去 **内容生成** 试写一篇，能正常开始输出文字。

### 新手常见错误

| 现象 | 可能原因 | 怎么办 |
|---|---|---|
| 测试连接失败 | Key 填错、复制多了空格、Key 没额度 | 重新复制 Key，确认平台余额 |
| 模型列表为空 | 平台接口没拉到模型 | 手动填写模型名，或点刷新/测试连接 |
| 点了保存还是不能写 | 只保存了 Key，没点设为当前使用 | 回到页面点 **设为当前使用** 再保存 |

---

## 6. 第二步：配置图片生成 API

这一步不是必须，但建议配置。否则文章可能没有封面图或配图。

### 进入位置

```text
设置 -> 图片生成API
```

### 最简单方案：先用 Picsum

如果你只是想先测试流程：

1. 找到 Picsum。
2. 设置为当前图片源。
3. 保存设置。

Picsum 不需要 Key，但图片是随机图，和文章主题可能不完全匹配。

### 更正式方案：用阿里百炼或魔搭

以阿里百炼为例：

1. 去阿里云百炼控制台创建 DashScope API Key。
2. 回到软件的 **图片生成API** 页面。
3. 找到阿里百炼对应配置。
4. 填入 API Key。
5. 选择图片模型。
6. 点击测试。
7. 成功后设为当前。
8. 点击保存设置。

---

## 7. 第三步：生成第一篇文章

### 进入位置

```text
内容生成
```

### 最简单写法

1. 在话题输入框里输入你想写的主题。

   例子：

   ```text
   新手做公众号如何坚持日更
   ```

2. 生成数量先选 **1**。
3. 完成后先选 **不操作** 或 **存草稿**。
4. 新手建议先打开 **过滤已写**，避免重复选题。
5. 如果你想快一点，可以打开 **极速模式**。
6. 点击 **开始生成**。

### 极速模式和正常模式怎么选

| 模式 | 适合谁 | 特点 |
|---|---|---|
| 极速模式 | 想快速试流程的新手 | 更快，但文章打磨少一点 |
| 正常模式 | 想要正式发文 | 更慢，但质量更好 |

第一次使用建议先用 **极速模式** 跑通流程。正式发文时再用正常模式。

### 生成过程中看哪里

- **实时写作**：看文章内容慢慢生成。
- **运行日志**：看当前执行到哪一步。
- **进度条**：看整体流程是否还在运行。

生成完成后，文章会进入 **文章库**。

## 8. 使用借鉴模式

借鉴模式适合你看到一篇好文章，想让 AI 学习它的结构、风格或角度，但不是照抄。

### 入口

在 **内容生成** 页面点击：

```text
借鉴模式
```

### 常用设置

| 设置 | 建议 |
|---|---|
| 参考链接 | 粘贴你想参考的文章链接 |
| 借鉴比例 | 新手建议 10% 到 30% |
| 模板分类 | 不确定就先不选 |
| 模板名称 | 不确定就随机 |

建议不要把借鉴比例开太高。正式发布前一定要人工检查，避免内容过于接近原文。

---

## 9. 在文章库查看和处理文章

### 进入位置

```text
文章库
```

这里可以做：

- 查看已生成文章。
- 搜索标题或关键词。
- 切换网格/列表视图。
- 刷新文章列表。
- 批量选择文章。
- 发布文章。
- 删除不需要的文章。
- 查看素材库。

### 新手推荐流程

1. 进入 **文章库**。
2. 找到刚生成的文章。
3. 打开预览，检查标题、正文、配图。
4. 如果不满意，回到内容生成重新生成，或使用优化功能。
5. 确认没问题后，再复制使用或发布到公众号草稿箱。

## 10. 配置微信公众号发布

如果你只想用软件写文章，然后手动复制到公众号后台，这一步可以先跳过。

如果你想一键上传草稿，需要配置微信公众号。

### 进入位置

```text
设置 -> 微信公众号
```

### 准备 AppID 和 AppSecret

1. 打开微信公众平台：https://mp.weixin.qq.com
2. 登录你的公众号。
3. 进入 **设置与开发 -> 基本配置**。
4. 复制 **开发者 ID（AppID）**。
5. 查看或重置 **AppSecret**，管理员扫码后复制保存。

注意：AppSecret 通常只显示一次，请保存好。

### 配置 IP 白名单

微信公众号接口要求你的电脑公网 IP 在白名单里。

1. 回到软件的 **微信公众号** 设置页。
2. 找到 **服务器出口 IP**。
3. 点击复制。
4. 回微信公众平台的 **基本配置 -> IP 白名单**。
5. 粘贴并保存。

如果你换了网络，比如从家里换到公司，公网 IP 可能变化，需要重新配置白名单。

### 在软件里填写

1. 填入 AppID。
2. 填入 AppSecret。
3. 填入作者名。
4. 新手建议勾选 **仅草稿**。
5. 点击 **测试连接**。
6. 成功后点击 **保存设置**。

### 发布建议

新手不要直接群发。建议流程：

```text
文章库 -> 发布 -> 上传到公众号草稿箱 -> 去公众号后台预览 -> 人工确认 -> 再发布
```

## 11. 使用选题中心找热点

### 进入位置

```text
选题中心 -> 聚合资讯
```

你可以点击 **一键获取热点**，软件会汇总多来源热点。

适合这样用：

1. 先获取热点。
2. 找一个适合你账号定位的话题。
3. 一键带入内容生成，或复制标题到内容生成。
4. 再让 AI 写文章。

不要什么热点都写。新手建议只选和自己账号定位有关的内容。

---

## 12. 使用运行中心做定时任务

### 进入位置

```text
运行中心 -> 自动化任务
```

它可以做到点自动写稿、保存到文章库，也可以配合公众号上传草稿。

新手使用前先确认：

- 大模型 API 已配置成功。
- 如果要配图，图片生成 API 已配置。
- 如果要发公众号，微信公众号已测试连接成功。
- 软件到点时需要保持运行。

### 新建定时任务

1. 点击 **新建任务**。
2. 填写创作话题。
3. 如果留空，系统会尝试自动抓热点。
4. 选择平台，当前建议优先用微信公众号。
5. 设置时间。
6. 选择是否生成配图。
7. 保存任务。

新手建议先让任务只生成文章并保存，不要一开始就自动群发。

## 13. 模板库怎么用

### 进入位置

```text
模板库
```

模板决定文章排版风格，比如标题样式、分割线、正文版式、重点强调样式等。

新手建议：

- 一开始不用管模板，先用默认模板。
- 等文章生成流程跑通后，再挑选适合自己账号风格的模板。
- 如果某篇文章排版不好，可以尝试重新套模板。

## 14. 常见问题

### 软件打不开

可能原因：

- WebView2 没安装或损坏。
- 杀毒软件拦截。
- 启动过程被异常中断。

可以尝试：

1. 运行 `修复WebView2.bat`。
2. 再运行 `启动小爆来咯.bat`。
3. 如果桌面窗口仍打不开，试试 `启动-浏览器.bat`。

### 生成文章一直失败

优先检查：

- 大模型 API 是否测试成功。
- Key 是否还有余额。
- 模型名是否可用。
- 网络是否能访问对应服务商。

### 生成很慢

可能是正常情况。正常模式会做更多分析、撰写、审核、优化和排版。

想先快速测试，可以开启：

```text
极速模式
```

### 没有配图

检查：

- 图片生成 API 是否配置。
- 是否设为当前。
- 是否保存设置。
- 图片平台 Key 是否有额度。

如果只是测试流程，可以先用 Picsum。

### 公众号测试失败

常见原因：

- AppID 或 AppSecret 填错。
- AppSecret 重置后软件里还是旧的。
- IP 白名单没有配置。
- 换网络后公网 IP 变了。

处理方式：

1. 重新复制 AppID 和 AppSecret。
2. 在软件里刷新服务器出口 IP。
3. 回微信公众平台更新 IP 白名单。
4. 保存后重新测试连接。

### 文章发布后看不到

先确认你选的是：

- 上传到草稿箱。
- 还是正式群发。

新手一般应该选择草稿箱。去公众号后台的草稿箱里查看。

## 15. 新手最稳的使用流程

每天发文可以按这个流程：

```text
1. 打开软件
2. 去选题中心看热点，或者自己输入选题
3. 内容生成里生成 1 篇文章
4. 到文章库检查内容
5. 不满意就重新生成或优化
6. 满意后上传到公众号草稿箱
7. 去公众号后台预览
8. 人工确认后发布
```

---

## 16. 安全提醒

请保护好你的 Key 和公众号密钥：

- 不要把 API Key 发给别人。
- 不要截图公开包含 Key 的页面。
- 不要把 `secrets/api_keys.yaml` 发到群里。
- AppSecret 泄露后请立刻去微信公众平台重置。
- 群发功能要谨慎使用，建议先走草稿箱。

---

## 17. 一句话总结

小白先记住这三个入口就够了：

```text
设置：配置 AI 和公众号
内容生成：写文章
文章库：查看、修改、发布文章
```

先跑通一篇，再慢慢学热点、模板、定时任务。不要一上来就开自动群发，稳一点，文章质量也更好。
