# 小爆来咯 · 功能详解手册（按模块 / 按钮）

> 这份手册逐个模块、逐个按钮讲解「小爆来咯」的界面功能，方便你在使用时随时查阅。
>
> 阅读约定：左侧是导航菜单，点击切换不同模块；带 \* 号的菜单（微信公众号、大模型 API）是必须配置项。部分菜单仅在特定模式下显示。
>
> 飞书提示：打开本文后点右上角「目录 / 大纲」图标，可按标题快速跳转。

---

## 1. 界面总览

软件主界面分三块：

- 左侧「导航菜单」：在各功能模块之间切换。
- 中间「主工作区」：当前模块的具体内容。
- 顶部「标题栏」：应用名、窗口最小化 / 最大化 / 关闭按钮。

左侧导航从上到下依次是（部分项随权限显示）：

- 工作台
- 内容生成
- 文章库
- 知识库
- 模板库
- 选题中心（含「聚合资讯」「资讯采集」两个子项）
- 能力中心
- 运行中心（含「自动化任务」「任务监控」两个子项）
- 素材中心
- 设置（含 7 个子面板 + IP 白名单）

---

## 2. 工作台

进入软件后的总览页，用来快速了解整体状态。

主要内容：

- 顶部标题「工作台」与版本徽标。
- 三块统计卡片：活跃话题数、今日生成文章、平均拟人得分，每张卡片下方有趋势标签（稳定 / 实时）。
- 中部可视化图表区。

按钮：

- 重置视角：把图表的缩放 / 旋转角度恢复到默认状态。

---

## 3. 内容生成

这是最核心的写作模块，输入选题后让 AI 写文章。

### 3.1 顶部话题区

- 话题输入框：填写你想写的主题，例如「春季护肤避坑指南」。留空则自动获取热搜话题。
- 自动配图 / 模板排版：两个状态徽标，表示当前会自动配图、自动套模板。

### 3.2 工具栏按钮

- 借鉴模式：打开「借鉴模式与模板参数」面板，可让 AI 参考一篇已有文章的风格来写。生成过程中无法切换。
- 运行日志 / 实时预览：切换中间输出区显示「日志」还是「正在生成的文章预览」。
- 重新加载：在借鉴模式下重新加载可选的参考文章列表，选择后会自动提取话题与参考链接。

### 3.3 借鉴模式参数

- 模板分类：限定使用哪一类模板，留空为「随机分类」。
- 模板名称：指定具体模板，留空为「随机模板」。
- 借鉴比例：控制对参考文章的借鉴程度。

### 3.4 实时预览区

- 等待生成 / 生成中：状态标签。
- 侧栏预览：在右侧栏打开手机 / 桌面样式预览。
- 滚动到底部：把预览内容滚到最新位置。
- 清空：清除当前预览内容。

### 3.5 运行日志区

- 复制日志：把全部运行日志复制到剪贴板。
- 清空日志：清除当前日志。
- 导出日志：把日志保存成文件，便于排查问题。

### 3.6 生成选项

- 生成数量：一次生成几篇文章。
- 完成后：生成完成后的动作，可选「不操作 / 存草稿 / 发布」。
- 极速模式：跳过深度审计与标题优化，加快生成（开启后「自动换模板」不可用）。
- 过滤已写：过滤最近 30 天内已处理过的话题，避免重复。
- 自动换模板：生成后自动更换模板并保存（保留配图；极速模式下不可用）。
- 合集模式：把话题作为固定前缀，每天生成不同子话题，标题格式为「话题：子话题」。

### 3.7 主操作按钮

- 开始生成：按当前选项启动写作。点击后会先「验证系统配置」，通过后进入抓取解析、策划大纲、撰写、终审、优化、配图排版等阶段，顶部进度条会逐步点亮。
- 生成过程中再次点击「开始生成」会放弃上次未完成的任务，从零重新开始。

## 4. 文章库

管理所有已生成的文章。

### 4.1 左侧栏

- 文章分类：按分类筛选文章。
- 素材库：查看生成文章时自动创建的配图。
- 整理助手：辅助批量整理文章的入口。

### 4.2 顶部工具条

- 搜索框：按标题 / 关键词搜索文章。
- 刷新：重新加载文章列表。
- 网格视图 / 列表视图：切换文章的展示方式。
- 批量操作：进入多选模式，配合下面两个按钮使用。
- 全选：选中当前列表的全部文章。
- 反选：反转当前的选择状态。

### 4.3 文章卡片按钮

每篇文章卡片上有一排小图标：

- 查看：预览文章完整内容。
- 编辑：进入编辑器修改文章。
- 复制到公众号：复制文章内容，粘贴到公众号后会尽量保留样式。
- 发布：把文章上传 / 发布到已配置的平台（当前主要支持微信公众号草稿箱）。
- 删除：删除该文章（不可撤销）。

文章状态标签：已发布 / 发布失败 / 未发布。

### 4.4 发布记录

在文章详情里可查看「发布记录」，列出每次发布的结果；没有记录时显示「暂无发布记录」。

---

## 5. 知识库（知识库与存储）

管理本地数据与 AI 的审美偏好学习结果。

主要内容：

- 数据根目录与环境：显示数据保存位置，标注「跨平台持久化」「运行正常」等状态。
- 审美偏好（DNA）：展示软件学到的「布局偏好 / 配色风格 / 氛围关键词」，未学习时显示「暂无」。
- 已投票文章池：参与过审美投票的文章集合。

按钮：

- 刷新数据：重新加载数据目录信息与审美偏好。
- 刷新（已投票文章池）：加载参与投票的文章列表。

---

## 6. 模板库

管理文章排版用的模板。

### 6.1 顶部

- 新建分类：创建一个新的模板分类。
- 搜索框：按模板名称搜索。
- 网格视图 / 列表视图：切换展示方式。

### 6.2 分类操作

鼠标悬停分类上可见：

- 编辑分类 / 重命名 / 删除分类：管理分类。删除有模板的分类会提示确认且不可撤销。

### 6.3 模板卡片按钮

- 编辑：修改模板内容。
- 重命名：修改模板名称。
- 复制：复制一份模板。
- 删除：删除该模板。
- 点击卡片：打开模板预览。

---

## 7. 选题中心

帮你找选题，分两个子页面。

### 7.1 聚合资讯（newshub）

- 分类下拉：按「全部分类 / 科技 / 编程 / 财经 / 社会 / 创业」等筛选热点。
- 一键获取最新热点（刷新）：抓取最新热点数据，旁边显示最近更新时间。
- 统计卡片：数据源、文章数、热点趋势、最近更新。
- 数据源配置 + 保存：选择启用哪些资讯来源并保存。

### 7.2 资讯采集（spider-manager）

- 采集器管理页面，用于配置和触发资讯抓取，结果可作为写作素材。

---

## 8. 能力中心（MCP 服务）

管理扩展能力（MCP 服务），给 AI 增加额外工具。

顶部统计：总计、已启用、运行中、错误。

按钮：

- 添加服务：弹出「添加 MCP 服务」表单，填写后点「添加」保存。
- 刷新：手动刷新服务状态（页面每 30 秒也会自动刷新一次）。
- 服务卡片内「保存」：保存该服务的配置修改。
- 安装提示里的命令：点击可复制到剪贴板。

---

## 9. 运行中心

定时与监控相关，分两个子页面。

### 9.1 自动化任务（定时发布）

到点自动写稿、保存到文章库，并可发布公众号；约每 1 分钟检查一次，需保持应用运行。当前仅微信公众号支持自动发布与连接检测。

- 统计：全部任务、今日执行。
- 刷新 / 刷新列表：重新加载任务与日志。
- 任务列表：显示编号、平台、下次执行、周期、状态、操作。
- 执行日志：显示时间、结果、说明、文章。

新建定时任务（点「新建」后弹窗）：

- 创作话题：可留空，到时自动抓热点。
- 发布平台 / 每次篇数 / 首次执行时间（必填） / 合集模式。
- 保存：创建任务；取消：放弃。

### 9.2 任务监控（swarm-monitor）

- 实时竞价与撮合、共识记忆节点等运行态可视化监控（高级 / 受限功能）。

---

## 10. 素材中心（预览画廊）

集中查看生成的图片素材与预览效果，便于挑选和管理配图。

---

## 11. 设置

左侧「设置」展开后有多个子面板。带 \* 的是必填项。

### 11.1 常规设置

- 主题模式：亮色 / 暗色主题，保存后立即生效。
- 窗口模式：标准窗口 / 最大化启动（部分模式需重启生效）。
- 页面设计器主题：跟随系统 / 默认主题。
- 发布平台：微信公众号 / 小红书 / 抖音 / 今日头条 / 百家号 / 知乎 / 豆瓣（当前仅微信公众号完整可用）。
- 文章格式：生成文件的默认格式（纯文本 TXT）。
- 模板分类 / 模板选择：默认使用的模板，可设为随机。
- 最大 / 最小搜索数、最少 / 最多字数：控制资料搜索量与文章长度。
- 文章配图数量：含封面，1–12 张，默认 3。
- 恢复默认 / 保存设置：还原或保存本页配置。

### 11.2 内容生成（维度化创意）

- 创意强度：0.7 保守 · 1.0 适中 · 1.5 激进。
- 最大维度数：同时参与改写的维度上限。
- 兼容性阈值：越高越要求维度之间风格协调。
- 全部展开 / 全部收起：展开或收起维度列表。
- 启用推荐维度：一键勾选官方推荐的维度组合。
- 恢复默认 / 保存设置。

### 11.3 选题来源（platforms）

- 配置写作选题的来源平台与开关。

### 11.4 微信公众号 \*

- 填写公众号的 AppID、AppSecret 等凭证。
- 添加凭证：支持配置多个公众号。
- 出口 IP 区域：显示本机出口 IP（公众号需把它加入 IP 白名单）。
  - 刷新出口 IP：重新获取当前 IP。
  - 复制 IP：点击复制到剪贴板。
- 保存：保存公众号配置。

### 11.5 大模型 API \*

- 配置写文章用的大模型服务（如 DeepSeek、通义千问、OpenRouter 等）。
- 每个提供商可填写 API Base、API Key、模型名称。
- 测试连接：验证 Key 是否可用。
- 设为当前使用：把某个配置设为默认。
- 添加自定义提供商：接入未预置的服务（如 Agnes AI）。
- 保存：保存 API 配置。

### 11.6 图片生成 API

- 配置封面图与正文配图用的图片服务。
- 与大模型 API 类似，可填写 Key、测试、保存。
- 不配置也能先试写文章，但可能没有配图或只用随机图。

### 11.7 页面设计

控制新生成文章的 HTML 排版样式，参数会注入新文章：

- 容器：最大宽度、左右外边距、页面背景色。
- 卡片：圆角、内边距、背景色、阴影 CSS。
- 排版：基础字号、行高倍数、标题字号倍数、正文颜色、标题颜色。
- 间距：Section 间距、元素间距。
- 配色：主色调、辅助色、高亮背景。
- 恢复默认 / 保存设置。

### 11.8 IP 白名单（受限显示）

- 白名单列表：显示备注、状态、创建 / 更新时间。
- 新增：填写 IP（如 1.2.3.4）和备注后「保存」。
- 刷新：重新加载列表。
- 每条记录可编辑 / 删除。

## 12. 标题栏与通用操作

- 窗口按钮：最小化、最大化 / 还原、关闭。
- 通知提示：操作成功 / 失败时右侧会弹出绿色或红色提示条。
- 大部分列表页都有「刷新」按钮，遇到数据不同步时先点它。

---

## 13. 新手最短上手路线

把核心模块串起来，最快跑通一篇文章：

1. 设置 → 大模型 API：填 Key，点「测试连接」，再「设为当前使用」并保存。
2. （可选）设置 → 图片生成 API：配好配图服务。
3. 内容生成：输入话题（或留空），点「开始生成」。
4. 文章库：查看生成结果，按需「编辑」。
5. 设置 → 微信公众号：填凭证、加 IP 白名单。
6. 文章库：点「复制到公众号」或「发布」。

> 想自动化，就到「运行中心 → 自动化任务」新建一个定时任务，到点自动写、自动发。
